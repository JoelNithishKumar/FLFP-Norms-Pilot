# FLFP-Norms-Pilot

**Misperception and Norms in Female Labor Force Participation: A Pilot Information Experiment**

This is the folder structure for the project. Step 2 will fill in:
- `data/raw/` — the three CSV data files (PLFS state panel, South Asia World Bank comparator, your Google Forms pilot responses)
- `R_scripts/` — the R analysis script
- `figures/`, `tables/` — generated by running the R script
- `article/` — the written article (will embed the generated figures)
- `presentation/` — the PowerPoint deck
- `docs/` — data source documentation and citations

Each `.gitkeep` file is just a placeholder so empty folders survive zip/extract and git. Safe to delete once each folder has real content in it.
