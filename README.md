# Misperception and Norms in Female Labor Force Participation

A pilot information experiment (N=9) benchmarked against a 181-country panel of female labor force participation rates (1990–2025), examining whether India's recent gains in female labor force participation are genuinely unusual and whether people's intuitions about them are well calibrated.

Read the full write-up: [`article/article.md`](article/article.md)

## How to use this repo

### 1. Reproduce the analysis

Everything in `figures/`, `tables/`, and `data/processed/` is generated by one script, using only base R (no packages to install).

1. Make sure the four files in `data/raw/` are present (they're already included in this repo).
2. Open [`R_scripts/01_main_analysis.R`](R_scripts/01_main_analysis.R). The first lines set:
   ```r
   PROJECT_ROOT <- "C:/dev/PreDoc_Projects/FLFP-Norms-Pilot"
   ```
   Update this to wherever you've cloned the repo (forward slashes, even on Windows).
3. Run the script:
   ```
   Rscript R_scripts/01_main_analysis.R
   ```
   or open it in RStudio and click **Run**.
4. The script will print its key results to the console (India's percentile ranks, ANOVA results, etc.) and (re)write every file in `figures/`, `tables/`, and `data/processed/`.

### 2. Read the write-up

[`article/article.md`](article/article.md) is a self-contained, fully cited research note. It renders natively on GitHub with all five figures embedded.

### 3. View the slides

[`presentation/FLFP_Norms_Pilot_Presentation.pptx`](presentation/FLFP_Norms_Pilot_Presentation.pptx) is a 13-slide deck covering the same material visually. Open it in PowerPoint, Keynote, or Google Slides. To rebuild it from scratch (e.g., after changing a figure):
```
cd presentation
npm install pptxgenjs sharp react react-dom react-icons
node build_presentation.js
```

## Data sources

1. International Labour Organization. *ILO Modelled Estimates database (ILOEST)*. ILOSTAT. https://ilostat.ilo.org/data/bulk/
2. World Bank. *World Development Indicators*, Indicator SL.TLF.CACT.FE.ZS. https://data.worldbank.org/indicator/SL.TLF.CACT.FE.ZS
3. Our World in Data. "Female labor force participation rate." https://ourworldindata.org/grapher/female-labor-force-participation-rates
4. Ravi, S., and M. Kapoor. 2024. "Female Labour Force Participation Rate: An Observational Analysis of the Periodic Labour Force Survey (PLFS) from 2017-18 to 2022-23." EAC-PM Working Paper Series EAC-PM/WP/34/2024. https://eacpm.gov.in/wp-content/uploads/2024/12/EACPM-WP-Female-LFPR-India.pdf
5. Ministry of Labour and Employment, Government of India. Press release on PLFS Annual Report results. https://www.pib.gov.in/PressReleasePage.aspx?PRID=2178389

Full citations, including the background literature referenced in the discussion, are in [`article/article.md`](article/article.md).
